# FitTrack Mobile App UI/UX Design

## Overview
FitTrack is a fitness mobile app signup flow designed in Figma.  
The goal of this project is to create a simple, clean, and user-friendly onboarding experience.

## Screens Designed
- Welcome Screen
- Create Account Screen
- Interactive Signup Flow Prototype

## Features
- Modern fitness app interface
- User-friendly signup process
- Input field designs
- Interactive navigation between screens

## Tool Used
- Figma
